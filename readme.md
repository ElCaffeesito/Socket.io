# Tarea sockets

Para instalar todas las dependencias utiliza el siguiente comando:

`npm i`

Para poder ejecutar todo en modo dev:

`npm run dev`

Para buildear el proyecto utilizar:

`npm run build`

Y para ejecutar el proyecto con:

`npm run start`
